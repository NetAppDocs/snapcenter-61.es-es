= Cómo acceder a la API REST de SnapCenter de forma nativa
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Puede acceder a la API REST de SnapCenter directamente utilizando cualquier lenguaje de programación que admita un cliente REST.  Las opciones de lenguaje más populares incluyen Python, PowerShell y Java.
