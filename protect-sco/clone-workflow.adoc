= Flujo de trabajo de clonación
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
El flujo de trabajo de clonación incluye la planificación, la ejecución de la operación de clonación y la supervisión de la operación.

Puedes clonar bases de datos por las siguientes razones:

* Para probar la funcionalidad que debe implementarse utilizando la estructura y el contenido de la base de datos actual durante los ciclos de desarrollo de la aplicación.
* Para rellenar almacenes de datos utilizando herramientas de extracción y manipulación de datos.
* Para recuperar datos que fueron eliminados o modificados por error.
