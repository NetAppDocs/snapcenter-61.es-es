= Desinstalar el paquete de complementos para Linux o AIX mediante CLI
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Puede desinstalar el paquete de complementos de SnapCenter para Linux o el paquete de complementos de SnapCenter para AIX mediante la interfaz de línea de comandos.

.Antes de empezar
* Asegúrese de haber eliminado los trabajos programados
* Asegúrese de que todos los trabajos en ejecución se hayan completado.


.Paso
Ejecute _/custom_location/ NetApp/snapcenter/spl/installation/plugins/uninstall_ para desinstalar.
