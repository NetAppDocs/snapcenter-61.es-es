= Qué puede hacer utilizando el complemento SnapCenter para la base de datos SAP HANA
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Cuando instala el complemento para la base de datos SAP HANA en su entorno, puede usar SnapCenter para realizar copias de seguridad, restaurar y clonar bases de datos SAP HANA y sus recursos.  También puede realizar tareas que respalden dichas operaciones.

* Agregar bases de datos.
* Crear copias de seguridad.
* Restaurar desde copias de seguridad.
* Copias de seguridad clonadas.
* Programar operaciones de respaldo.
* Supervisar operaciones de copia de seguridad, restauración y clonación.
* Ver informes de operaciones de copia de seguridad, restauración y clonación.
