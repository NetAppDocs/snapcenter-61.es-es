= Cancelar las operaciones de copia de seguridad del complemento SnapCenter para Microsoft SQL Server
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Puede cancelar operaciones de copia de seguridad que estén en ejecución, en cola o que no respondan.  Cuando se cancela una operación de copia de seguridad, el servidor SnapCenter detiene la operación y elimina todas las instantáneas del almacenamiento si la copia de seguridad creada no está registrada en el servidor SnapCenter .  Si la copia de seguridad ya está registrada en SnapCenter Server, no revertirá la instantánea ya creada incluso después de que se active la cancelación.

.Antes de empezar
* Debe iniciar sesión como administrador de SnapCenter o propietario del trabajo para cancelar las operaciones de restauración.
* Puede cancelar únicamente las operaciones de registro o de copia de seguridad completa que estén en cola o en ejecución.
* No es posible cancelar la operación una vez iniciada la verificación.
+
Si cancela la operación antes de la verificación, la operación se cancelará y no se realizará la operación de verificación.

* Puede cancelar una operación de copia de seguridad desde la página Monitor o desde el panel Actividad.
* Además de utilizar la GUI de SnapCenter , puede utilizar cmdlets de PowerShell para cancelar operaciones.
* El botón *Cancelar trabajo* está deshabilitado para las operaciones que no se pueden cancelar.
* Si seleccionó *Todos los miembros de este rol pueden ver y operar en objetos de otros miembros* en la página Usuarios\Grupos al crear un rol, puede cancelar las operaciones de respaldo en cola de otros miembros mientras usan ese rol.


.Pasos
Ejecute una de las siguientes acciones:

|===
| Desde... | Acción 


 a| 
Página de monitorización
 a| 
. En el panel de navegación izquierdo, seleccione *Monitor* > *Trabajos*.
. Seleccione el trabajo y seleccione *Cancelar trabajo*.




 a| 
Panel de actividades
 a| 
. Después de iniciar el trabajo de respaldo, seleccioneimage:../media/activity_pane_icon.gif["icono del panel de actividad"] en el panel Actividad para ver las cinco operaciones más recientes.
. Seleccione la operación.
. En la página Detalles del trabajo, seleccione *Cancelar trabajo*.


|===
.Resultado
La operación se cancela y el recurso vuelve al estado anterior.  Si la operación que canceló no responde en el estado de cancelación o ejecución, debe ejecutar la `Cancel-SmJob -JobID <int> -Force` cmdlet para detener forzosamente la operación de copia de seguridad.
