= Definir una estrategia de restauración y recuperación para los recursos de MySQL
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Debe definir una estrategia antes de restaurar y recuperar su base de datos para poder realizar operaciones de restauración y recuperación con éxito.


NOTE: Sólo se admite la recuperación manual de la base de datos.

.Pasos
. Determinar las estrategias de restauración compatibles con los recursos MySQL agregados manualmente
. Determinar las estrategias de restauración compatibles con las bases de datos MySQL detectadas automáticamente
. Decide el tipo de operaciones de recuperación que deseas realizar.
