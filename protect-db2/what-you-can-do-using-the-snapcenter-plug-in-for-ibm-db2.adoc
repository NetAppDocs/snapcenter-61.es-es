= Qué puede hacer utilizando el complemento SnapCenter para IBM Db2
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Cuando instala el complemento para IBM Db2 Database en su entorno, puede utilizar SnapCenter para realizar copias de seguridad, restaurar y clonar bases de datos IBM Db2 y sus recursos.  También puede realizar tareas de apoyo a dichas operaciones.

* Agregar bases de datos.
* Crear copias de seguridad.
* Restaurar desde copias de seguridad.
* Copias de seguridad clonadas.
* Programar operaciones de respaldo.
* Supervisar operaciones de copia de seguridad, restauración y clonación.
* Ver informes de operaciones de copia de seguridad, restauración y clonación.
