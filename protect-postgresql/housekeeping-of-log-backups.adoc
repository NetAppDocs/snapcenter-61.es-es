= Cómo gestiona SnapCenter el mantenimiento de las copias de seguridad de datos
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapCenter administra el mantenimiento de las copias de seguridad de datos en los niveles del sistema de almacenamiento y del sistema de archivos.

Las instantáneas del almacenamiento primario o secundario y sus entradas correspondientes en el catálogo de PostgreSQL se eliminan según la configuración de retención.
