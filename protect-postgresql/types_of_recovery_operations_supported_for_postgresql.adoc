= Tipos de operaciones de recuperación compatibles con clústeres de PostgreSQL
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapCenter le permite realizar diferentes tipos de operaciones de recuperación para clústeres PostgreSQL.

* Recuperar el clúster hasta el estado más reciente
* Recuperar el clúster hasta un punto específico en el tiempo
+
Debe especificar la fecha y hora para la recuperación.



SnapCenter también proporciona la opción Sin recuperación para clústeres PostgreSQL.
